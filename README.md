# HTTP Debugging Notes

This repository contains practical notes and troubleshooting scenarios related to web debugging using browser DevTools and HTTP analysis.

## Topics Covered
- HTTP status codes
- Request and response analysis
- Browser DevTools
- Network tab debugging
- Frontend vs backend issue identification
- Basic API troubleshooting

## Practical Scenarios
- Login request succeeds but user data is missing
- API returns empty responses
- Identifying failed requests using status codes
- Analyzing request headers and payloads

## Tools Used
- Chrome DevTools
- Browser Network Tab
- SQL basics for backend data validation
